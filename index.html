What is an Memory Paper Wallet?

Memory Paper Wallet is a customizable printable paper wallet generator using a brain wallet passphrase hardened with scrypt to make address generation both memory and time-intensive.

Main Features

  • Bitcoin segwit paper wallets
  • Ethereum, Solana, Monero, Oxen, Litecoin, EOS, and Ardor paper wallet graphics
  • User changable background for easy customization. (Gifts, Bdays, Weddings, Promotions, etc.)
  • Generates multiple crypto currencies with a username passphrase combination
  • Identicon integration for visual representation of your address.
  • Scrypt hardened to make brute force guessing almost impossible. Address generation is both memory and time-intensive.
  • Multi-Key paper wallets using Shamir Secret Sharing Scheme. Recontruct secret with passguardian.com or Android Shamir Secret Decoder
  • URL parameters user, pass, and lvl (i.e /?user=my_user&pass=my_pass&lvl=12345)

Whats the story behind this?

We created this to introduce Bitcoin to clients, friends, and family in the form or printable wallets. There wasn't any customizable paper wallet generator, which is why we created this one.

Warnings and Considerderations

  • PLEASE, DO NOT PUT YOUR LIFE SAVINGS ON THESE ADDRESSES. Get a hardware wallet for that.
  • Test wallets by sending a little first, then send it back out to verify its working!
  • Each "set" of generated addresses should not be shared with other people.
